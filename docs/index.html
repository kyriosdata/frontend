Missão

Facilitar a ambientação com os fundamentos da construção de aplicações web.

Avaliação

Precisamos aferir se a missão foi atingida, o que depende de presenças e de uma nota final.

A nota é resultado de um trabalho prático, cuja definição é assunto do primeiro encontro.

Ementa (programa)

A disciplina tem como foco três linguagens. Seguiremos uma ordem natural: HTML 5, CSS 3 e JavaScript.

A expectativa é que o trabalho prático seja realizado, ao menos parcialmente, durante os encontros. Ou seja, você aprende enquanto "faz a prova".
